## Basic install

<Tabs>
  <Tab title="pip">
    ```bash
    pip install orxhestra
    ```
  </Tab>
  <Tab title="uv">
    ```bash
    uv add orxhestra
    ```
  </Tab>
</Tabs>

## With an LLM provider

<Tabs>
  <Tab title="OpenAI">
    ```bash
    pip install orxhestra[openai]
    ```
  </Tab>
  <Tab title="Anthropic">
    ```bash
    pip install orxhestra[anthropic]
    ```
  </Tab>
  <Tab title="Google Gemini">
    ```bash
    pip install orxhestra[google]
    ```
  </Tab>
  <Tab title="MCP">
    ```bash
    pip install orxhestra[mcp]
    ```
  </Tab>
</Tabs>

<Info>
  **Python >= 3.10** is required.
</Info>
